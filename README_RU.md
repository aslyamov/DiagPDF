# DiagPDF — Генератор PDF с шахматными диаграммами

Конвертирует PGN / FEN / EPD файлы в профессиональные PDF-страницы формата A4 с шахматными диаграммами,
используя семейство шрифтов Chess Alpha DG.

## Скачать

Последнюю версию **DiagPDF.exe** можно получить в разделе [Releases](../../releases) — Python не нужен.

---

## Возможности

- Чтение файлов `.pgn`, `.fen`, `.epd`
- 7 макетов страницы: 1–4 колонки, до 20 диаграмм на страницу
- 4 шрифта доски: AlphaDG, LeipzigDG, CondalDG, KingdomDG
- Индикатор хода: квадрат □■, круг ○●, треугольник △▲
- Авто-переворот доски при ходе чёрных
- Координаты доски (опционально)
- Гибкие подписи диаграмм через шаблон: `{number}`, `{event}`, `{white}`, `{black}`, `{date}`, `{comment}`
- Нотационные линии под диаграммами: пустые или пронумерованные (1–5 строк)
- **Раздел ответов** в конце документа с фигурными обозначениями (шрифты Hastings / Zurich / Linares), 1 или 2 колонки
- **Кликабельные ссылки**: заголовок диаграммы → ответ, номер ответа → диаграмма
- **Ссылки на анализ Lichess**, встроенные в символ хода
- Верхний / нижний колонтитул с переменными `{page}`, `{total}`, `{chapter}`
- Поддержка глав через тег `[Chapter "..."]` в PGN
- Кириллица и Unicode в заголовках (через системные шрифты)
- GUI (Windows, Tkinter) + полноценный CLI
- Двуязычный интерфейс: English / Русский

## Установка

### Вариант A — Исполняемый файл (Windows, Python не нужен)

1. Скачайте `DiagPDF.exe` из раздела [Releases](../../releases)
2. Положите в любую папку и запустите

```bash

```

## Использование GUI

Запуск интерфейса:
```bash
python fen2rtf.py --gui
# или
DiagPDF.exe --gui
```

| Поле | Описание |
|------|----------|
| **Входной файл** | Нажмите `…`, чтобы выбрать `.pgn`, `.fen` или `.epd` файл |
| **Выходной PDF** | Заполняется автоматически; нажмите `…`, чтобы изменить |
| **Верхний / нижний колонтитул** | Текст с `{page}`, `{total}`, `{chapter}` |
| **Макет** | Пресет сетки страницы (колонки × строки) |
| **Шрифт** | Шрифт доски (рекомендуется AlphaDG) |
| **Размер шрифта доски** | 0 = авто-подбор; или введите конкретное значение в пт |
| **Символ** | Форма индикатора хода |
| **Линии** | Нотационных линий под каждой диаграммой (0–5), пустые или пронумерованные |
| **Ориентация** | Авто / Белые внизу / Чёрные внизу |
| **Показывать координаты** | Включить/отключить буквы и цифры по краям доски |
| **Шаблон заголовка** | Шаблон для подписей диаграмм — нажмите `?` для списка переменных |
| **Ссылки Lichess** | Встроить ссылку на анализ в символ хода |
| **Добавить раздел ответов** | Добавить раздел с решениями в конец документа |
| **Заголовок раздела ответов** | Название раздела ответов |
| **Колонки ответов** | 1 или 2 колонки в разделе ответов |
| **Шрифт фигур** | Шрифт для обозначений фигур в ответах (рекомендуется Zurich) |

Нажмите **Создать PDF**, затем **Открыть PDF** для просмотра результата.

## Справочник CLI

```bash
python fen2rtf.py input.pgn [опции]
```

| Опция | По умолчанию | Описание |
|-------|-------------|----------|
| `-o OUTPUT` | `<input>.pdf` | Путь к выходному PDF |
| `-l, --layout N` | `2` | Пресет макета 0–6 |
| `-f, --font NAME` | `AlphaDG` | Шрифт доски |
| `--font-size N` | `0` | Размер шрифта в пт (0 = авто) |
| `--no-coords` | — | Скрыть координаты доски |
| `--flip` | — | Перевернуть доску (чёрные внизу) |
| `--no-auto-flip` | — | Отключить авто-переворот при ходе чёрных |
| `--symbol NAME` | `square` | `square` / `circle` / `triangle` |
| `--lines N` | `0` | Нотационных линий на диаграмму |
| `--lines-numbered` | — | Пронумерованные линии (1. ___ ___) |
| `--header TEXT` | имя файла | Текст верхнего колонтитула |
| `--footer TEXT` | `{page}` | Текст нижнего колонтитула |
| `--title-template TPL` | `{number} {comment}` | Шаблон подписи |
| `--lichess-link` | — | Встроить ссылки Lichess |
| `--answers` | — | Добавить раздел ответов |
| `--answers-title TEXT` | `Solutions` | Заголовок раздела ответов |
| `--answers-cols N` | `1` | Колонки ответов (1 или 2) |
| `--figurine-font NAME` | `Zurich` | Шрифт фигур для ответов |
| `--version` | — | Вывести версию и выйти |
| `--gui` | — | Запустить GUI |

## Советы по формату PGN

### Подписи диаграмм

DiagPDF читает данные позиции из тегов PGN и первого комментария.
Используйте шаблон заголовка, чтобы составить нужную подпись:

```
--title-template "{number}. {event} ({date})"
--title-template "{number} {comment}"
```

### Главы

Добавьте тег `[Chapter "Название главы"]` перед группой партий.
Используйте `{chapter}` в колонтитуле для отображения текущей главы:

```pgn
[Chapter "Тактика — Связка"]
[Event "1.1 Пример"]
[FEN "..."]

1. Kg7! 1-0
```

### Раздел ответов

Ходы из тела партии PGN становятся ответами.
Комментарии `{...}` и коды NAG `$1` автоматически удаляются.
Варианты в скобках `(2...Ke8 3.Rb8#)` сохраняются.

## Шрифты

|------|-------|
| `AlphaDG.ttf` | Основной (VDMX патчится автоматически при первом запуске) |

| `CondalDG.ttf` | Конденсированный стиль |
| `KingdomDG.ttf` | Kingdom стиль |
| `ZurichFigurine.TTF` | Фигурная нотация — сбалансированный (по умолчанию) |
| `HastingsFigurine.TTF` | Фигурная нотация — классический, детальный |
| `LinaresFigurine.TTF` | Фигурная нотация — упрощённый, современный |

Патченые копии (`*_patch.ttf`) создаются автоматически при первом запуске.
