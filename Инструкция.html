DiagPDF v0.2.0
Генератор PDF с шахматными диаграммами

Программа конвертирует шахматные файлы PGN / FEN / EPD в профессиональные PDF-страницы с диаграммами, используя шахматные шрифты Chess Alpha DG.

Установка

1
Установите шрифты
Откройте папку Fonts, выделите все файлы (Ctrl+A), нажмите правой кнопкой мыши и выберите «Установить» (или «Установить для всех пользователей»).
Шрифты нужны для корректного отображения диаграмм в PDF.
2
Запустите DiagPDF.exe
Дважды щёлкните DiagPDF.exe. Если Windows показывает предупреждение, нажмите «Подробнее» → «Выполнить в любом случае».

Использование

  1. Нажмите «…» рядом с полем Входной файл и выберите файл .pgn, .fen или .epd.
  2. При необходимости измените путь к выходному PDF.
  3. Настройте параметры (макет, шрифт, символ хода и т.д.) — или оставьте значения по умолчанию.
  4. Нажмите «Создать PDF».
  5. После завершения нажмите «Открыть PDF» для просмотра.

Основные настройки

ПолеОписание
МакетКоличество диаграмм на странице (1–4 колонки)
ШрифтВид диаграммы: AlphaDG, LeipzigDG, CondalDG, KingdomDG
Размер шрифта доски0 = авто-подбор; или конкретное значение в пт
СимволИндикатор хода: квадрат □■, круг ○●, треугольник △▲
ЛинииНотационные линии под диаграммой (0–5 строк, пустые или пронумерованные)
Шаблон заголовкаПодпись диаграммы: {number}, {event}, {white}, {black}, {date}, {comment}
Раздел ответовДобавляет решения задач в конец документа
Колонки ответов1 или 2 колонки в разделе ответов
Шрифт фигурДля обозначений фигур в ответах (Zurich, Hastings, Linares)
Ссылки LichessВстраивает ссылку на анализ в символ хода

Главы в PGN

Добавьте тег [Chapter "Название"] перед группой партий. Используйте {chapter} в колонтитуле для отображения текущей главы:

[Chapter "Тактика — Связка"]
[Event "1.1"]
[SetUp "1"]
[FEN "..."]

1. Ng7! 1-0
Ответы: ходы из тела партии PGN автоматически попадают в раздел ответов. Комментарии {...} и оценки $1 убираются; варианты (...) сохраняются.

Примеры PDF

В папке Example/ находятся готовые PDF с демонстрацией разных настроек:

  • ex1 — 2 колонки, AlphaDG, без линий
  • ex2 — 2 колонки, AlphaDG, 2 линии
  • ex3 — 2 колонки, LeipzigDG, пронумерованные линии
  • ex4 — 2 колонки + раздел ответов 1 колонка (шрифт Zurich)
  • ex5 — 3 колонки, CondalDG + ответы 2 колонки (шрифт Hastings)
  • ex6 — 4 колонки, KingdomDG, треугольный индикатор

DiagPDF v0.2.0
Chess Diagram PDF Generator

Converts PGN / FEN / EPD chess files to professional A4 PDF pages with diagrams, using the Chess Alpha DG font family.

Setup

1
Install the fonts
Open the Fonts folder, select all files (Ctrl+A), right-click and choose Install (or Install for all users).
The fonts are required for correct diagram rendering in PDF.
2
Run DiagPDF.exe
Double-click DiagPDF.exe. If Windows shows a SmartScreen warning, click More info → Run anyway.

Usage

  1. Click … next to Input file and choose a .pgn, .fen, or .epd file.
  2. Change the output PDF path if needed (auto-filled by default).
  3. Adjust the settings (layout, font, to-move symbol, etc.) or leave the defaults.
  4. Click Generate PDF.
  5. Click Open PDF to view the result.

Settings reference

FieldDescription
LayoutNumber of diagrams per page (1–4 columns)
FontBoard style: AlphaDG, LeipzigDG, CondalDG, KingdomDG
Board font size0 = auto-fit; or a specific pt value
SymbolTo-move indicator: square □■, circle ○●, triangle △▲
LinesNotation lines below diagram (0–5, plain or numbered)
Title templateDiagram caption: {number}, {event}, {white}, {black}, {date}, {comment}
Add answers sectionAppends solutions at the end of the document
Answer columns1 or 2 columns in the answers section
Figurine fontPiece symbols in answers (Zurich, Hastings, Linares)
Lichess linksEmbeds analysis link in the to-move symbol

Chapters in PGN

Add a [Chapter "Name"] tag before a group of games. Use {chapter} in the header/footer to display the current chapter:

[Chapter "Tactics — Pin"]
[Event "1.1"]
[SetUp "1"]
[FEN "..."]

1. Ng7! 1-0
Answers: moves in the PGN game body automatically become the answers. Comments {...} and NAG codes $1 are stripped; variations (...) are preserved.

Example PDFs

The Example/ folder contains ready-made PDFs demonstrating different settings:

  • ex1 — 2 columns, AlphaDG, no lines
  • ex2 — 2 columns, AlphaDG, 2 notation lines
  • ex3 — 2 columns, LeipzigDG, numbered lines
  • ex4 — 2 columns + answers section, 1 column (Zurich figurine font)
  • ex5 — 3 columns, CondalDG + answers 2 columns (Hastings figurine font)
  • ex6 — 4 columns, KingdomDG, triangle indicator

Archive contents
